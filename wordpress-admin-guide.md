# Руководство для владельца сайта

Теперь ваш сайт связан с WordPress. Вы можете управлять услугами и отзывами через админ-панель.

## Как добавить новую услугу

1. Зайдите в админ-панель WordPress.
2. В меню слева выберите **Услуги** -> **Добавить новую**.
3. **Заголовок**: Введите название услуги (например, "Замена масла").
4. **Редактор**: Описание услуги не используется на карточке, но может использоваться для детальной страницы в будущем.
5. **Изображение записи (Featured Image)**: Загрузите фото для карточки.
6. **ACF поля** (под редактором):
   - **Цена**: Введите цену (например, "от 500₽").
   - **Короткое описание**: Текст, который будет показан на карточке услуги (1-2 предложения).
7. Нажмите **Опубликовать**.

## Как добавить отзыв

1. В меню выберите **Отзывы** -> **Добавить новую**.
2. **Заголовок**: Можно ввести имя клиента (для поиска в админке).
3. **Редактор**: Введите текст отзыва.
4. **ACF поля**:
   - **Оценка**: Число от 1 до 5.
   - **Имя клиента**: Имя, которое будет отображаться.
   - **Автомобиль**: Марка авто (например, "Kia Rio").
5. Нажмите **Опубликовать**.

## Обновление сайта

После внесения изменений в WordPress, сайт нужно пересобрать (если это не настроено автоматически):

1. Сообщите разработчику о внесении изменений.
2. Или, если у вас настроен авто-деплой, изменения появятся через несколько минут.
