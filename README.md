# AutoElectric - Astro Project

Лендинг для автоэлектрика в Краснодаре, созданный на Astro фреймворке.

## 🐛 Исправленные баги из исходного HTML

1. ✅ **Исправлен невалидный inline style на карте** (строки 801-803 в code.html)
2. ✅ **Исправлена опечатка** "Краснодарcкий" → "Краснодарский"
3. ✅ **Добавлена анимация** `animate-fade-in-up` в Tailwind конфигурацию
4. ✅ **Удалены дублирующие импорты** Material Symbols шрифта
5. ✅ **Добавлен переключатель темной темы** с localStorage
6. ✅ **Добавлены обработчики кнопок** и модальное окно формы
7. ✅ **Улучшена доступность** - использованы aria-label вместо data-alt
8. ✅ **Добавлена валидация формы** с обработкой отправки

## 🚀 Запуск проекта

```bash
# Установка зависимостей
npm install

# Запуск dev сервера
npm run dev

# Сборка для продакшена
npm run build
```

```
project-auto/
├── src/
│   ├── components/
│   │   ├── Header.astro          # Шапка с темной темой
```
